# Thumbnailer

A simple CLI tool to create image thumbnails. Made with [Pluto](https://github.com/phenopolis/pluto).

Currently only JPEG is supported. More formats are planned.

## Usage

```
thumbnailer *.jpg  --widths 100,200 --heights 300,400
```
